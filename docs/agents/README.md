# Agent Runbooks

These runbooks are written for Claude, Codex, and other coding agents that operate this repository.

Use them as task-specific instructions:

- `setup-dify-host.md`: install or repair the always-on Dify host.
- `setup-cloudflare.md`: publish the Remote MCP and gateway through Cloudflare Tunnel and Access.
- `setup-admin-command.md`: configure the `rag-access-email` allowlist command.
- `user-onboarding.md`: add or remove connector user access and run acceptance tests.
- `troubleshooting.md`: diagnose connection, search, ingestion, and permission failures.
- `safety.md`: review checklist for secrets, public docs, and access boundaries.

Do not copy secrets into these files. All examples must stay generic.
